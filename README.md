# Flamework Template

This project uses a pre-configured template you can use for your projects that use [Flamework](https://fireboltofdeath.dev/docs/flamework/).

For further information on how to use Flamework in your project, please see [here](https://fireboltofdeath.dev/docs/flamework).

# Setup and run

1. Install the project dependencies and run the build command in watch mode (hot relod)

```bash
bun install

bun run watch
```

2. [Sync changes to Roblox Studio](https://marketplace.visualstudio.com/items?itemName=evaera.vscode-rojo)

3. Done!
